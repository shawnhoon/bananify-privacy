Bananify Privacy Policy

Last Updated: December 23, 2025

Privacy-First Design: Bananify is built with privacy as a core principle. All data is stored locally in your browser. We only send selected text to Google's Gemini API for image generation - nothing else.

Overview

Bananify is a Chrome extension that generates visual infographics from selected webpage text using Google's Gemini AI. This privacy policy explains how we handle your data and what information is collected, stored, and transmitted.

Data Collection

Bananify collects minimal data necessary for its core functionality:

  • Selected Text Content: When you select text on a webpage and generate an infographic, we collect the text you selected. This text is sent to Google's Gemini API for processing and stored locally with the generated infographic for your reference.
  • Page Metadata: We capture the URL and title of the webpage where you selected text. This is stored locally with your infographic to provide context.
  • Authentication Information:
    • For BYOK (Bring Your Own Key) mode: We store your Google Gemini API key locally, encrypted using Web Crypto API (AES-GCM).
    • For backend mode: We use Google Sign-In OAuth tokens for authentication. These tokens are stored locally and used only for API authentication.
  • Usage Statistics: We track generation counts and timestamps locally for cost estimation purposes. This data never leaves your browser.
  • Generated Infographics: All generated infographic images are stored locally in your browser's IndexedDB.

What We Do NOT Collect:

  • Personally identifiable information (names, addresses, email addresses)
  • Browsing history or web pages you visit
  • Location data or IP addresses
  • Financial or payment information
  • Health information
  • Personal communications
  • User activity monitoring (clicks, mouse position, keystrokes)
  • Telemetry or analytics data

Data Storage

All data collected by Bananify is stored locally in your browser using Chrome's storage APIs:

  • Chrome storage.local: Settings, preferences, encrypted API keys, and usage statistics
  • IndexedDB: Generated infographic images and metadata
  • Chrome storage.session: Temporary session data (encryption passwords, if used)

No data is stored on external servers. Everything remains in your browser and can be deleted at any time.

Data Transmission

Bananify transmits data only in the following circumstances:

  • To Google Gemini API: When you generate an infographic, the text you selected and your configured prompts are sent to Google's Gemini API at https://generativelanguage.googleapis.com/*. This is the only external data transmission. All communication is encrypted via HTTPS.
  • To Google Sign-In: If you use backend mode, authentication requests are sent to Google's OAuth servers for sign-in. We do not receive or store your Google account information beyond what's necessary for authentication.

We do not transmit data to any other third-party services, analytics platforms, or advertising networks.

Third-Party Services

Google Gemini API

Bananify uses Google's Gemini API to generate infographics. When you generate an infographic:

  • The text you selected is sent to Google's servers
  • Your configured prompts and settings are included
  • Google processes this data and returns a generated image

Google's use of this data is governed by Google's Privacy Policy. We recommend reviewing Google's privacy policy to understand how they handle data sent to their API.

Google Sign-In

If you use backend mode, authentication is handled through Google Sign-In. Google's use of authentication data is governed by their privacy policy. We do not receive or store your Google account email or other personal information.

Data Security

  • Encryption: API keys are encrypted using Web Crypto API (AES-GCM) before storage
  • Local Storage: All data remains in your browser - no external servers store your information
  • HTTPS: All external communication is encrypted via HTTPS
  • No Plaintext Storage: Sensitive data (API keys) is never stored in plaintext

Your Rights and Control

You have complete control over your data:

  • Delete Data: You can delete all extension data at any time through the extension settings
  • Export Data: You can download all generated infographics from the gallery
  • Revoke Access: You can remove your API key or sign out at any time
  • Uninstall: Uninstalling the extension removes all locally stored data

Permissions Explained

Bananify requests the following permissions and uses them as follows:

  • activeTab: Access selected text on the current webpage when you explicitly use the extension
  • storage: Save settings, encrypted API keys, and generated infographics locally
  • scripting: Extract clean text content from web pages
  • notifications: Notify you when infographic generation is complete
  • sidePanel: Display generation status and progress
  • contextMenus: Add "Generate Infographic" option to right-click menu
  • clipboardWrite: Copy generated infographics to clipboard when you click "Copy"
  • tabs: Access tab URL and title for source tracking
  • identity: Google Sign-In authentication for backend mode
  • Host Permission (Google Gemini API): Required to send selected text to Google's API for image generation

Data Retention

Data is retained as follows:

  • Generated Infographics: Stored locally until you delete them or uninstall the extension
  • Settings and Preferences: Stored locally until you change or delete them
  • API Keys: Stored locally (encrypted) until you remove them
  • Usage Statistics: Stored locally until you clear history or uninstall

You can delete any or all of this data at any time through the extension settings.

Children's Privacy

Bananify is not intended for children under 13 years of age. We do not knowingly collect personal information from children. If you believe we have inadvertently collected information from a child, please contact us to have it removed.

Changes to This Privacy Policy

We may update this privacy policy from time to time. The "Last Updated" date at the top of this page indicates when changes were made. We encourage you to review this policy periodically. Continued use of Bananify after changes constitutes acceptance of the updated policy.
